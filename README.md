🍵 Teacup AI Agent - 基于 Spring AI Alibaba 的智能助手框架

一个功能丰富的 AI Agent 项目，集成 RAG(检索增强生成)、文件操作、PDF 生成、终端操作等核心功能。支持通义千问模型 (Qwen-plus/Qwen3.5-plus)，提供持久化聊天记忆、文档加载与向量存储、多工具调用等能力。

- 🛠️ 丰富工具集：文件操作、PDF 生成、资源下载、终端命令、网页搜索/爬取
- 🔌 Spring AI Alibaba 集成：支持 Qwen 系列模型
- 📝 完善的测试覆盖

🏗️ 技术栈: Java 21 + Spring Boot 3.x + Spring AI Alibaba + Maven



🍵 Teacup AI Agent - Intelligent Assistant Framework powered by Spring AI Alibaba

A feature-rich AI Agent project integrating RAG (Retrieval-Augmented Generation), file operations, PDF generation, terminal control, and more. Supports Qwen models (Qwen-plus/Qwen3.5-plus) with persistent chat memory, document loading, vector store, and multi-tool capabilities.

✨ Key Features:
- 🧠 RAG Enhanced: Custom document loader, vector store configuration, query augmenter
- 💾 Persistent Chat Memory: File-based Kryo serialization
- 🛠️ Rich Tool Set: File ops, PDF generation, resource download, terminal commands, web search/scraping
- 🔌 Spring AI Alibaba Integration: Qwen series model support
- 📝 Comprehensive test coverage

🏗️ Tech Stack: Java 21 + Spring Boot 3.x + Spring AI Alibaba + Maven
